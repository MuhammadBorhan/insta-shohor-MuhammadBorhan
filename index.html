Q1: Difference Between Local Storage vs Session Storage.

Local Storage:

Local storage is a way to store data on the user's computer. It allows storing key/value pairs in a web browser and it stores data without an expiration date. We can clear this data using local storage methods. Local Storage capacity 10mb. Local storage is accessible from any window.

window.localStorage: stores data with no expiration date.

Session Storage:

Session storage is a popular choice for storing data in the browser. This enables developers to preserve and restore different values. Session storage only holds data for a specific session. Data is cleared when the user closes the browser window. Session storage capacity 5Mb. Session storage is accessible from the same tab. Session storage location browser only.

window.sessionStorage : stores data for one session (data is lost when the browser tab is closed).
